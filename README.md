# Assignment: Data Analysis with Python

## Problem Statement
This project analyzes two small datasets - the Iris flower dataset and the Wine dataset - to compare their numerical attributes and identify patterns in biological and chemical measurements. The goal is to understand similarities and differences between plant morphological features (iris) and wine chemical compositions, potentially revealing insights into how different domains' data can be characterized statistically.

## Datasets Used
1. Iris Dataset: Contains measurements of sepal length, sepal width, petal length, and petal width for three species of iris flowers.
2. Wine Dataset: Contains chemical analysis results of wines grown in the same region in Italy, including alcohol content, malic acid, ash, and other chemical properties.
